Avatar Upload Example

  • STEP 1: Before Image Selection
  • STEP 2: After Image Selection
  • STEP 3: During Image Processing
  • STEP 4: After Image Processing
  • STEP 5: Complete

Before Image Selection

Before a user uploads their avatar, if they hover over the placeholder, it will show the upload button, if they click it will allow them to browse for an image to upload.

After Image Selection

After the user has seleted their avatar image it will appear in the avatar placeholder window. This window will allow the user to zoom in and out, and drag the image around to place where they like.

During Image Processing

After the user clicks on the save button, the image will be resized and cropped on the client side before it is automatically uploaded to the server.

After Image Processing

After processing, the window will show a green background with a check mark giving the user some feedback that their avatar was uploaded to the server successfully, then will fade out after a few seconds to show their final avatar.

After Image Upload Complete

If the user hovers over their avatar again it will show the upload button again to start the process over if they wish to replace their current avatar.
